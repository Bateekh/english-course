## Features

- 365 daily lessons per level (12 months)
- Text-to-speech pronunciation (normal + slow speed)
- Word-by-word breakdown
- Arabic translation and tips for every lesson
- Progress tracking saved in your browser
- Backup and restore progress across devices
- Mobile-friendly design
